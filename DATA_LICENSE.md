# Data License Status

**Current status: pending provenance and source-license verification.**

The MIT `LICENSE` in this repository applies only to the validation scripts and original repository documentation. It does **not** automatically grant redistribution rights for the dataset contexts.

Before public distribution:

1. recover the source of every context,
2. record the source URL, revision/version and retrieval date,
3. verify the source license and attribution requirements,
4. determine whether the context text may be redistributed,
5. choose and document a compatible dataset license.

Until these checks are complete, this package should be treated as an internal release candidate and should not be represented as a legally cleared public dataset.
